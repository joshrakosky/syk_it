# Product Images

Place product thumbnail images here.

## Tile Mate 4 Pack Images

- `tile-mate-black.jpg` - Black Tile Mate 4 Pack thumbnail
- `tile-mate-white.jpg` - White Tile Mate 4 Pack thumbnail

## Image Requirements

- Recommended size: 800x800px or larger (square aspect ratio)
- Format: JPG, PNG, or WebP
- File size: Keep under 500KB for fast loading

## Adding Images

1. Save images to this `public/images/` directory
2. Update the database with the correct paths:
   - `/images/tile-mate-black.jpg`
   - `/images/tile-mate-white.jpg`

Or use full URLs if hosting images elsewhere (Supabase Storage, CDN, etc.)
